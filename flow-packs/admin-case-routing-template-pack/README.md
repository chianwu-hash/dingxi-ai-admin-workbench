# 行政案件類型路由模板流程包

## 這是什麼？

這個流程包教你建立一張「案件類型 -> 必讀規則 -> 禁止誤用 -> 預設輸出」的路由表，讓 AI 行政助理先判斷案件類型，再套用正確規則。

## 適合誰？

- 行政人員
- 組長或主任
- 想建立 AI 行政工作臺的人
- 想避免 AI 對所有案件套同一格式的人

## 你會做出什麼成果？

完成後會得到：

- 一張案件類型路由表
- 幾種常見行政案件的輸出格式
- 一份風險升級與人工確認清單

## 使用前需要準備什麼？

- 你常處理的行政案件類型
- 每種案件的基本規則
- 哪些情況需要主管確認

## 最快開始方式

1. 複製 `02-template/case-routing-table-template.md`。
2. 填入 3 到 5 種常見行政案件。
3. 用 `03-demo-data/sample-cases.md` 練習分類。

## 常見錯誤

- 沒先分類就直接套格式。
- 不同案件共用同一段輸出。
- 忽略個資、主管、正式紀錄等高風險情境。
