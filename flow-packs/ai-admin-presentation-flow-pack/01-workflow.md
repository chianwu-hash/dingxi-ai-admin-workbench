# AI 行政簡報製作流程

## 流程總覽

```text
brief
-> slide outline
-> image prompt
-> image generation
-> review
-> fixed overlay
-> export
-> SOP retrospective
```

## 重點

- 不要只看成品 PDF，要讓使用者理解製作流程。
- 一般使用者從網站提示詞開始；flow-pack 是 AI 或維護者備查材料。
- 先確認主題、聽眾、使用場合、可用素材與不可公開內容。
- 正式校徽、頁碼、固定識別不交給影像模型生成。
- 圖像生成後要檢查文字、品牌、事實與視覺遮擋。
- 完成後要請 AI 回頭整理流程，留下可交接、可修正的小 SOP。

## 人與 AI 分工

- AI 協助：整理 brief、提出缺資料問題、產生投影片大綱、列出圖像需求與審查清單。
- 人確認：事實、數字、姓名、照片授權、個資、校內未公開資料、對外分享界線。
- 團隊留存：把可重複使用的 brief、提示詞、檢查點與修正方式寫回 SOP。
