# SOP 回寫提示詞模板

完成一份行政簡報後，可以把這段提示詞貼給 AI，請它回頭整理成下次可重複使用的小 SOP。

```text
請把剛剛完成這份行政簡報的過程，整理成一份下次可以重複使用的小 SOP。

請包含：
- 這個 SOP 適合什麼情境
- 需要準備哪些資料
- 操作步驟
- AI 可以幫忙的部分
- 人一定要檢查的部分
- 下次可以直接複製的提示詞

請用行政同仁看得懂的語氣，不要寫成技術文件。
請保留必要的安全提醒，例如不要放個資、不要公開內部資料、正式校徽與固定識別要用 overlay 或正式素材處理。
```

## 使用方式

1. 先完成 brief、大綱、圖像提示詞與審查清單。
2. 把實際使用過的提示詞、遇到的問題、人工修正方式一起提供給 AI。
3. 請 AI 產生 SOP 後，由人補上校內規定、責任分工與最後確認點。
4. 下次製作類似簡報時，先照 SOP 做，再把不穩定的地方修得更明確。

## 人工確認點

- SOP 是否只使用可公開、可分享的範例。
- 是否清楚寫出 AI 只能協助整理，不能替人做正式決定。
- 是否有固定輸出格式，避免每次結果差太多。
- 是否列出缺資料時應該先問的問題。
- 是否提醒正式校徽、頁碼與固定識別不要交給影像模型自由生成。
