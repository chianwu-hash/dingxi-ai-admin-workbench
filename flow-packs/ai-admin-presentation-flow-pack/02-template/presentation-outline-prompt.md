# 投影片大綱提示詞

把簡報 brief 貼給 AI 後，可以用這段提示詞請 AI 產生第一版投影片大綱。

```text
請依照我提供的簡報 brief，產生一份可以人工審查的投影片大綱。

請輸出成表格，欄位包含：
- 頁碼
- 頁面標題
- 這一頁的目的
- 主要內容
- 需要的素材或圖像
- 人工確認事項

請遵守：
- 每一頁只放一個主要目的。
- 不要補寫我沒有提供、也無法合理推論的事實。
- 如果缺少重要資料，請在人工確認事項中標示。
- 不要要求影像模型生成正式校徽、頁碼、印章、浮水印或固定識別。
- 如果簡報可能對外分享，請提醒檢查個資、內部資料、照片授權與去識別化。

完成後，請先不要進入視覺設計；請等我確認大綱順序、重點和安全界線。
```
