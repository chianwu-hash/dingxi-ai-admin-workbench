# 文件轉 Markdown 流程包

## 這是什麼？

這個流程包整理「把文件變成 AI 看得懂的乾淨文字」所需的流程、指令備查、檢查清單與安全提醒。

一般使用者建議先從網站單元頁開始，複製提示詞交給 AI 助手處理；本流程包主要給 AI 助手、教學者與維護者備查。

## 適合誰？

- 行政文件整理者
- 研習資料整理者
- 教師
- 想把文件交給 AI 摘要、分類或改寫的人

## 你會做出什麼成果？

完成後會得到：

- 一份 Markdown 或 Text 文件
- 一份轉換品質檢查清單
- 一個可交給 AI 後續處理的乾淨文字版本

## 使用前需要準備什麼？

- 要轉換的文件
- 可本機操作或可引導你操作的 AI 助手
- 可檢查輸出內容的文字編輯器

## 最快開始方式

1. 先把文件路徑或資料夾路徑交給 AI 助手。
2. 請 AI 檢查可用工具，必要時安裝或用白話步驟引導你完成。
3. 轉成 Markdown 或 Text 後，用品質清單檢查標題、表格、亂碼與漏字。
4. 把轉換後的乾淨文字交給 AI 摘要、整理待辦或改寫成公告。

## 常見錯誤

- 轉換後中文斷行錯誤。
- 表格欄位跑掉。
- 頁碼、頁首頁尾混入正文。
- 掃描 PDF 沒有文字層，輸出幾乎空白。

## 延伸應用

- 會議資料摘要
- 研習講義整理
- 公告初稿整理
- 表格資料轉成 AI 可讀文字
