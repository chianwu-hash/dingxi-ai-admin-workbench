# 文件轉 Markdown 流程包

## 這是什麼？

這個流程包教你把 PDF、Word、PowerPoint 或其他行政文件轉成 AI 容易閱讀的 Markdown / Text。

## 適合誰？

- 行政文件整理者
- 研習資料整理者
- 教師
- 想把文件交給 AI 摘要、分類或改寫的人

## 你會做出什麼成果？

完成後會得到：

- 一份 Markdown 或 Text 文件
- 一份轉換品質檢查清單
- 一個可交給 AI 後續處理的乾淨文字版本

## 使用前需要準備什麼？

- 要轉換的文件
- `doc2md-toolkit` 或 MarkItDown
- 可檢查輸出內容的文字編輯器

## 最快開始方式

1. 先用 `doc2md "檔案.pdf"` 轉出 Markdown。
2. 用品質清單檢查標題、表格、亂碼與漏字。
3. 把轉換後的 Markdown 交給 AI 摘要或整理。

## 常見錯誤

- 轉換後中文斷行錯誤。
- 表格欄位跑掉。
- 頁碼、頁首頁尾混入正文。
- 掃描 PDF 沒有文字層，輸出幾乎空白。

## 延伸應用

- 會議資料摘要
- 研習講義整理
- 公告初稿整理
- 表格資料轉成 AI 可讀文字
