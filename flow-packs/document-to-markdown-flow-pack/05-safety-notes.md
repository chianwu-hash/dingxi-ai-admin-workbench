# 安全注意事項

- 轉換前先確認文件是否含個資或未公開資料。
- 公開示範請使用假文件。
- 真實公文、會議資料、學生或家長資料不可直接放入公開流程包。
- 轉換後的 Markdown 仍可能含敏感資料，分享前需再次檢查。
- NotebookLM 可作為大量或複雜文件的雲端閱讀理解路線，但只適合沒有個資、機密或未公開資料的文件。
- 使用 NotebookLM 前，使用者必須自行確認文件可以上傳到 Google 服務；若不確定，請保留在本機處理或標記為需人工確認。
- 未來若使用 CDP 腳本輔助 NotebookLM，只能在使用者已登入、已上傳來源並明確同意後協助貼提示詞、擷取結果與保存 Markdown；不得自動登入、不得自行選取敏感文件、不得保存 cookies、profile 或 session。
