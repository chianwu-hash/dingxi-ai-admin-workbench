# Obsidian 勾選確認清單流程

## 流程總覽

```text
AI 整理待確認事項
-> 轉成 Markdown checkbox
-> 承辦人在 Obsidian 勾選
-> 回覆勾選結果給 AI
-> AI 依勾選結果整理下一步
```

## 步驟 1：整理待確認事項

先請 AI 把需要確認的內容整理成清單。

每一項只能是一個決策，例如：

- 是否分享某個流程包
- 是否要公開某份範例
- 是否需要先去識別化

## 步驟 2：轉成 checkbox

使用這種格式：

```markdown
- [ ] A. 同意分享文件轉 Markdown 流程包。
  - 影響：勾選後會放入教學網站下載中心。
```

## 步驟 3：在 Obsidian 勾選

把 Markdown 存成 `.md`，在 Obsidian 打開後直接勾選。

## 步驟 4：回覆 AI

可以使用簡短格式：

```text
確認 A/C；B 暫緩；D 不分享
```

## 步驟 5：AI 整理結果

AI 依照勾選結果更新整理稿、製作計畫或下一步清單。
