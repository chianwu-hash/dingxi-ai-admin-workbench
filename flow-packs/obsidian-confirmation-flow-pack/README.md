# Obsidian 勾選確認清單流程包

## 這是什麼？

這個流程包教你把 AI 整理出的待確認事項，轉成可在 Obsidian 或 Markdown 編輯器中勾選的確認清單。

## 適合誰？

- 行政承辦人
- 組長或主任
- 需要把 AI 建議交回人工確認的人
- 想保留可追蹤決策紀錄的人

## 你會做出什麼成果？

完成後會得到：

- 一份 Markdown checkbox 確認清單
- 一組勾選後回覆 AI 的格式
- 一個可重複使用的人機確認流程

## 使用前需要準備什麼？

- Obsidian，或任何能編輯 Markdown 的工具
- 一組需要確認的事項
- 一個願意把 AI 建議先勾選確認、再執行的工作習慣

## 最快開始方式

1. 複製 `02-template/checklist-template.md`。
2. 把每個待確認事項改成單一 checkbox。
3. 勾選後用 `02-template/reply-patterns.md` 的格式回覆 AI。

## 常見錯誤

- 一個 checkbox 放了兩個以上決策。
- 沒寫清楚勾選後下一步。
- 把內部敏感決策原文放進公開示範。

- 分享項目篩選
