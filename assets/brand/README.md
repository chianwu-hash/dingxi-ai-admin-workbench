# Brand Assets

This folder currently supports two modes:

## Internal Dingxi Mode

The working copy may contain authorized Dingxi Elementary School brand assets, such as:

- `dingxi-crest.png`
- `four-mascots-reference.png`
- `admin-workbench-banner.webp`
- `admin-action-banner.webp`

These are for local or school-internal use.

## Public Template Mode

Public GitHub exports should use placeholder assets:

- `placeholder-school-crest.svg`
- `placeholder-hero-banner.svg`
- `placeholder-action-banner.svg`

The public repo may include the internal network brand source pointer in `brand/brand-source.json`, but should not include official crest image files.

If the internal source is unavailable, do not generate, redraw, imitate, or trace the official school crest. Keep placeholders.
