Module 2

文件轉 Markdown

把 PDF、Word、PPT 轉成 AI 可讀的乾淨文字，再做摘要、分類或改寫。

為什麼要先轉文字？

AI 處理行政文件時，最怕輸入不乾淨。Markdown 讓標題、清單、表格與段落更容易被穩定理解。

下載流程包指令速查

最小流程

取得文件 -> 轉成 Markdown -> 檢查標題、表格、亂碼、漏字 -> 交給 AI 摘要或整理。

可跟做

使用假會議資料練習，不需要真實公文。

可檢查

流程包提供品質檢查清單，避免轉換後漏字或亂碼。

可延伸

可用於會議資料、研習講義、公告草稿與表格整理。
